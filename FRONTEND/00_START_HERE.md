## 🎨 Customize in 2 Minutes

### Change Primary Color
File: `tailwind.config.js` (line 6)
```javascript
primary: '#0084ff',  // Change this to your color
```

### Change App Logo
File: `src/components/Sidebar.jsx` (line 11)
```jsx
<span className="text-white font-bold text-lg">💬</span>  // Change emoji
```

### Add New Conversations
File: `src/data/chats.js`
```javascript
{
  id: 10,
  name: 'Your Chat',
  avatar: 'https://...',
  // ... other properties
}
```
